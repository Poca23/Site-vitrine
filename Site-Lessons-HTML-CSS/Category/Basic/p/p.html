Tutorial p

Simple balise p

Définition et utilisation :
La < p> balise définit un paragraphe.
Les navigateurs ajoutent automatiquement une seule ligne vide avant et après chaque < p> élément.
Astuce : utilisez CSS pour styliser les paragraphes.

Prise en charge :

Attributs globaux :
La < p> balise prend également en charge les attributs globaux en HTML.

Attributs d'évènement :
La < p> balise prend également en charge les attributs d'événement en HTML.
