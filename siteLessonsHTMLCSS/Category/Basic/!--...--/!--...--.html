Les commentaires "!--...--!" ne sont pas affichés dans un serveur

Définition et utilisation :
La balise comment est utilisée pour insérer des commentaires dans le code source. Les commentaires ne sont pas affichés dans les navigateurs.
Vous pouvez utiliser des commentaires pour expliquer votre code, ce qui peut vous aider lorsque vous modifierez le code source ultérieurement.
Ceci est particulièrement utile si vous avez beaucoup de code.

Remarque :
Les deux barres obliques à la fin de la ligne de commentaire (//) constituent le symbole de commentaire JavaScript. Cela empêche JavaScript d'exécuter la balise -->.

Attributs standards :
La balise de commentaire ne prend en charge aucun attribut standard.
